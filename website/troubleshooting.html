Troubleshooting

I can't install any version later than 1.1.1

Google recently stopped supporting 32-bit apps. This means that we needed to update the app from the old 32-bit version to 64 bits so that we can keep up to date with newer phones.

Alternatively, you might have a phone that uses an x86 processor architecture instead of an ARM architecture. We only support ARM architectures as it covers the vast majority of the smartphone market, and maintaining separate versions is more work than our small team can handle.

To find out what processor architecture your phone uses, check this guide


I get an error saying 'Application appears to be invalid'

Have you made sure that you uninstalled the original instagram app first? If you already have dfinstagram installed, try uninstalling it then reinstalling the newer version.
